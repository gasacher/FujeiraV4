# Fujeira V4

Sitio web desarrollado para la materia SEO, basado en la optimización de un proyecto existente previamente desarrollado.

## Objetivo

Aplicar buenas prácticas de SEO on-page, estructura semántica y optimización técnica sobre un sitio real.

## Mejoras realizadas

- Optimización de etiquetas `<title>` y `<meta description>`
- Uso correcto de headings (`h1`, `h2`, etc.)
- Implementación de estructura semántica HTML (`header`, `main`, `section`, `footer`)
- Incorporación de atributos `alt` en imágenes relevantes
- Configuración de `robots.txt`
- Creación y ajuste de `sitemap.xml`
- Mejora de accesibilidad en elementos visuales
- Diseño responsive utilizando Bootstrap

## Tecnologías utilizadas

- HTML5
- CSS3 / SCSS
- Bootstrap
- JavaScript

## Control de versiones

Se utilizó Git para registrar los cambios realizados durante el proceso, reflejando las distintas etapas de mejora del sitio.
